# Simple Unix Shell

Реализовал базовый функционал задачи, по работе с встроенными функциями и поддержкой конвейеров


### Базовые команды:
```bash
> pwd
/home/user/project

> echo Hello World
Hello World

> cd /tmp
> pwd
/tmp
>ps 
...

> kill <pid>
```

### Пример запуска конвейера:
```bash
> open -a "Google chrome" | open -a "Docker"
```
